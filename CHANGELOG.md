# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0

Initial release.

### Features

- MCP tool `loopy` — blocks AI execution until user responds
- Desktop feedback window (Tauri + Vue 3)
  - Frameless, transparent, always-on-top
  - Expand/collapse with smooth animation
  - Multi-session tab support
  - Image paste and file attachment
  - Chat history with copy support
- Auto-reply with configurable timeout and PUA methodology
- Multi-agent support with session isolation
- WebSocket-based communication between MCP server and GUI
- Session persistence via localStorage
